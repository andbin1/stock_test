👥 用户如何使用？（告诉他们）

Windows 用户：
   1. 解压 ZIP 文件
   2. 双击运行"安装依赖.bat"（仅首次）
   3. 双击运行"启动应用.bat"
   4. 浏览器访问 http://localhost:5000
   5. 查看"用户使用手册.md"了解详细功能

Mac/Linux 用户：
   1. 解压 ZIP 文件
   2. pip install -r requirements_release.txt
   3. python app_with_cache.py
   4. 浏览器访问 http://localhost:5000

   • 其他开发用的临时文件

   • 所有 .py 核心文件

   • .bat 启动脚本
