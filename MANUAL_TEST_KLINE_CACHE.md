## 常见问题排查

### Q1: K线还是在变化怎么办？
**可能原因**:
1. 浏览器缓存了旧版本的HTML
2. 文件没有保存成功

**解决方案**:
```bash
# 强制刷新浏览器（清除缓存）
Ctrl + F5  (Windows)
Cmd + Shift + R  (Mac)

# 检查文件是否正确修改
cd "D:\ai_work\stock_test"
grep "cachedKLineData" templates\parameters_config.html
# 应该看到6行结果
```

### Q2: 点击"重新生成K线"按钮没反应
**可能原因**:
1. JavaScript错误
2. 函数未正确定义

**解决方案**:
```javascript
// 打开浏览器开发者工具（F12）
// 查看Console是否有错误
// 手动执行测试
regenerateKLine();
// 应该看到K线重新生成
```

### Q3: 切换策略后图表还显示旧数据
**可能原因**:
1. 缓存未清空
2. 定时器延迟

**解决方案**:
```javascript
// 打开Console，手动清空缓存并重绘
window.cachedKLineData = null;
drawKLineChart();
```
