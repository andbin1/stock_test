### 测试用例1: 基本持仓计算
```
输入: 初始资金 100k, 占比 20%, 股价 50元
计算: floor(100000 × 0.2 ÷ 50 ÷ 100) × 100 = 400手
验证: ✅ 通过
```

### 测试用例2: 滑点影响
```
输入: 原价 100元, 滑点 1%
买入价: 100 × (1 + 0.01) = 101元
卖出价: 100 × (1 - 0.01) = 99元
验证: ✅ 正确应用
```

### 测试用例3: 手续费影响
```
输入: 交易金额 10000元, 费率 万1 (0.1%)
费用: 10000 × 0.001 = 10元
验证: ✅ 正确计算
```

### 测试用例4: 完整流程
```
初始资金: 100k
占比: 1/5 (20%)
股价: 50元
滑点: 1%
费率: 万1

持仓: 400手
买入价: 50 × 1.01 = 50.5元
卖出价: 50 × 0.99 = 49.5元
买入成本: 50.5 × 400 × 0.001 = 20.2元
卖出成本: 49.5 × 400 × 0.001 = 19.8元
总费用: 40元
实际收益: (49.5-50.5) × 400 - 40 = -440元

验证: ✅ 完整流程正确
```
