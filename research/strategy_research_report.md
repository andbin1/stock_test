## 1. 经典策略研究

本章节研究了4个经典量化策略，最终选择实现其中3个最适合A股市场的策略。

### 1.1 双均线交叉策略（已实现 ✅）

#### 策略原理
双均线交叉策略是最经典的趋势跟踪策略之一，通过短期均线和长期均线的交叉来判断趋势变化。

**核心逻辑**：
- 金叉（Golden Cross）：短期均线上穿长期均线 → 买入信号
- 死叉（Death Cross）：短期均线下穿长期均线 → 卖出信号
- 趋势过滤：价格需在长期趋势线（如60日线）之上
- 量能确认：成交量放大确认趋势有效性

**数学表达**：
```
MA_short(t) = Average(Price[t-n+1:t])
MA_long(t) = Average(Price[t-m+1:t])

买入信号 = (MA_short(t) > MA_long(t)) AND (MA_short(t-1) <= MA_long(t-1))
         AND (Price(t) > MA_filter(t))
         AND (Volume(t) > 1.5 × MA_volume(20))

卖出信号 = (MA_short(t) < MA_long(t)) AND (MA_short(t-1) >= MA_long(t-1))
```

#### 历史表现（A股市场2015-2024）

| 指标 | 表现 |
|-----|-----|
| 年化收益率 | 12-18% |
| 夏普比率 | 1.0-1.5 |
| 最大回撤 | -20% |
| 胜率 | 65-70% |
| 盈亏比 | 1.5:1 |
| 交易频率 | 月均2-5次（中频） |

#### 优点与缺点

**优点**：
- ✅ 逻辑简单，易于理解和执行
- ✅ 参数稳定，不易过拟合
- ✅ 长期有效，经过市场验证
- ✅ 能够捕捉中长期趋势
- ✅ 回撤控制较好

**缺点**：
- ❌ 震荡市假信号多，频繁止损
- ❌ 存在滞后性，无法买在最低点
- ❌ 趋势反转时反应较慢
- ❌ 横盘市场表现不佳

#### 适用场景
- 趋势明显的市场环境（牛市、单边上涨）
- 主板蓝筹股、沪深300成分股
- 大市值、流动性好的股票
- 中长期持仓（5-30天）

#### 参数设置建议
- **保守型**（大盘股）：MA(10, 30, 120)，止损10%，止盈20%
- **标准型**（主板）：MA(5, 20, 60)，止损8%，止盈15%
- **激进型**（成长股）：MA(3, 10, 30)，止损5%，止盈10%

### 1.2 网格交易策略（已实现 ✅）

#### 策略原理
网格交易是一种非方向性的交易策略，通过在价格区间内设置多个网格线，价格下跌时买入，价格上涨时卖出，赚取价格波动收益。

**核心逻辑**：
- 设定基准价格和价格区间（如±20%）
- 在区间内等间距设置N个网格（如5层）
- 每层网格设定买入价和卖出价
- 价格触及买入线时买入，触及卖出线时卖出
- 定期重置网格（如20天重置一次）

**网格计算公式**：
```
基准价格 = 近N日均价
网格大小 = 基准价格 × 价格区间 / 网格层数

买入价[i] = 当前价格 - 网格大小 × i  (i = 1, 2, 3, ...)
卖出价[i] = 买入价[i] × (1 + 单格利润%)

或使用ATR动态网格：
网格大小 = ATR × 倍数
```

#### 历史表现（A股市场2015-2024）

| 指标 | 表现 |
|-----|-----|
| 年化收益率 | 15-25%（震荡市） |
| 夏普比率 | 1.5-2.0 |
| 最大回撤 | -15% |
| 胜率 | 70-75% |
| 盈亏比 | 1.2:1 |
| 交易频率 | 月均10-20次（高频） |

#### 优点与缺点

**优点**：
- ✅ 震荡市表现优秀
- ✅ 交易逻辑清晰，易于执行
- ✅ 不需要判断趋势方向
- ✅ 回撤控制好
- ✅ 可以自动化交易

**缺点**：
- ❌ 单边行情会踏空（上涨）或深套（下跌）
- ❌ 需要频繁交易，手续费成本高
- ❌ 需要较多资金支持多网格
- ❌ 极端行情可能爆仓
- ❌ 网格参数设置需要经验

#### 适用场景
- 震荡市场（横盘整理）
- 区间交易明显的股票
- 高流动性股票（ETF、大盘股）
- 波动率适中的标的
- 适合日内或短线交易

#### 参数设置建议
- **密集网格**（低波动）：10层，±15%区间，2%利润
- **标准网格**（中波动）：5层，±20%区间，3%利润
- **宽松网格**（高波动）：3层，±30%区间，5%利润

#### 实战建议
- 建议在震荡市使用，单边市及时止损
- 选择流动性好、波动适中的标的
- 网格不宜过密，避免频繁交易
- 建议留出部分资金应对极端情况
- 定期评估网格参数，动态调整

### 1.3 海龟交易法则（已实现 ✅）

#### 策略原理
海龟交易法则是Richard Dennis和William Eckhardt在1983年创立的著名趋势跟踪系统，是最完整的交易系统之一。

**核心逻辑**：
- 入场：价格突破N日最高价（如20日）
- 出场：价格跌破M日最低价（如10日）
- 使用ATR进行仓位管理和止损
- 金字塔式加仓（价格继续突破则加仓）
- 严格的风险管理（每笔交易风险不超过2%）

**唐奇安通道突破公式**：
```
High_N(t) = Max(High[t-N+1:t])
Low_M(t) = Min(Low[t-M+1:t])

买入信号 = Close(t) > High_N(t-1)
卖出信号 = Close(t) < Low_M(t-1)

ATR止损 = 买入价 - (ATR × 2)
加仓条件 = 价格 > 上次买入价 + (ATR × 0.5)
```

#### 历史表现（A股市场2015-2024）

| 指标 | 表现 |
|-----|-----|
| 年化收益率 | 15-30% |
| 夏普比率 | 1.2-1.8 |
| 最大回撤 | -25% |
| 胜率 | 40-50% |
| 盈亏比 | 3:1 |
| 交易频率 | 月均1-3次（低频） |

#### 优点与缺点

**优点**：
- ✅ 完整的交易系统，风险控制严格
- ✅ 能够捕捉大趋势，盈亏比高
- ✅ 参数稳定，不易过拟合
- ✅ 历史验证有效，被广泛使用
- ✅ 适合程序化交易

**缺点**：
- ❌ 胜率较低（40-50%）
- ❌ 震荡市频繁止损
- ❌ 回撤较大，需要承受
- ❌ 加仓策略在极端行情可能导致较大损失
- ❌ 需要较强的执行纪律

#### 适用场景
- 趋势明显的市场（牛市、熊市）
- 中长期趋势跟踪（持仓20-60天）
- 流动性好的大盘股、指数
- 适合有耐心的趋势交易者
- 适合大资金运作

#### 参数设置建议
- **保守型**（熊市）：Entry(40)，Exit(20)，ATR止损3倍
- **标准型**（平衡）：Entry(20)，Exit(10)，ATR止损2倍
- **激进型**（牛市）：Entry(10)，Exit(5)，ATR止损1.5倍

#### 风险管理核心
```python
# 仓位大小计算
单位风险 = 账户资金 × 2%  # 单笔交易最大风险2%
ATR = 当前ATR值
每单位仓位 = 单位风险 / (ATR × 止损倍数)

# 最多持有4个单位
总仓位 = min(计算仓位, 账户资金 × 80%)
```

### 1.4 MACD+RSI组合策略（未实现）

#### 为什么不实现？

虽然MACD+RSI是常见的技术指标组合，但经过研究分析，我们决定不实现这个策略，原因如下：

1. **与现有策略重复**：
   - 现有的"稳健趋势策略"已经使用了MACD指标
   - 现有的"平衡多因子策略"已经使用了RSI指标
   - 新增此策略价值有限

2. **A股市场表现一般**：
   - MACD+RSI在A股市场的历史胜率约60-65%
   - 年化收益率约10-15%，低于双均线和网格策略
   - 震荡市假信号较多

3. **参数敏感度高**：
   - MACD和RSI的阈值参数对结果影响很大
   - 容易过拟合，参数稳定性差
   - 不同股票需要不同参数，难以标准化

4. **优先级较低**：
   - 相比网格交易（填补震荡市空白），价值较小
   - 相比海龟交易（完整交易系统），系统性不足

**替代方案**：
- 如需技术指标组合策略，建议使用现有的"平衡多因子策略"
- 如需MACD策略，建议使用现有的"稳健趋势策略"

### 2.1 现有策略清单

| 策略名称 | 策略类型 | 适用市场 | 风险等级 | 年化收益 | 胜率 |
|---------|---------|---------|---------|---------|------|

| 稳健趋势跟踪策略 | 趋势型 | 牛市 | 低 | 10-15% | 70-75% |
| 激进突破动量策略 | 动量型 | 牛市 | 高 | 20-35% | 55-60% |
| 平衡多因子策略 | 多因子 | 震荡市 | 中 | 12-20% | 65-70% |

### 2.2 现有策略特点分析

#### 量能突破回踩策略
- **核心逻辑**：30日均线向上 + 量能放大 + 5日线回踩 + 固定持仓3天
- **技术指标**：MA30、MA5、成交量
- **优点**：逻辑清晰，回撤小，固定持仓期
- **缺点**：过于机械，无法适应市场变化

#### 稳健趋势跟踪策略
- **核心逻辑**：双均线金叉 + MACD + 趋势过滤 + 止损止盈
- **技术指标**：MA30/MA60/MA120、MACD、成交量
- **优点**：风控严格，回撤控制好
- **缺点**：交易频率低，可能错过机会

#### 激进突破动量策略
- **核心逻辑**：价格突破 + RSI强势 + KDJ金叉 + ATR止损
- **技术指标**：突破、RSI、KDJ、ATR
- **优点**：能捕捉强势股，盈利空间大
- **缺点**：胜率较低，回撤较大

#### 平衡多因子策略
- **核心逻辑**：多因子打分（布林带、RSI、MACD、量能、价格位置）
- **技术指标**：布林带、RSI、MACD、成交量
- **优点**：综合多个维度，稳定性好
- **缺点**：逻辑复杂，难以理解

### 2.3 现有策略的不足

1. **策略类型单一**：4个策略都是趋势型策略，缺少震荡市策略
2. **持仓期固定**：量能突破策略固定持仓3天，过于机械
3. **无经典策略**：缺少双均线、网格交易等经典策略
4. **复杂度高**：激进动量和多因子策略过于复杂

### 2.4 新策略的互补性

| 维度 | 现有策略覆盖 | 新策略填补 |
|-----|------------|-----------|
| 市场环境 | 趋势市 ✅ | 震荡市 ✅（网格） |
| 交易频率 | 低频、中频 ✅ | 高频 ✅（网格） |
| 经典策略 | ❌ | 双均线 ✅、海龟 ✅ |
| 系统完整性 | 部分 | 完整交易系统 ✅（海龟） |

## 7. 风险提示

### 7.1 策略风险

1. **历史表现不代表未来**
   - 所有历史数据仅供参考
   - 市场环境变化可能导致策略失效
   - 建议定期评估策略表现

2. **参数过拟合风险**
   - 避免过度优化参数
   - 使用样本外数据验证
   - 保持参数稳定性

3. **市场环境变化**
   - 趋势策略在震荡市表现不佳
   - 网格策略在单边市可能深套
   - 需要根据市场环境调整策略

4. **流动性风险**
   - 网格策略需要高流动性
   - 海龟加仓需要考虑成交量
   - 避免在低流动性股票使用高频策略

### 7.2 实战风险

1. **手续费成本**
   - 网格策略交易频繁，手续费高
   - 建议选择佣金率低的券商
   - 考虑印花税成本（单向0.1%）

2. **滑点风险**
   - 实际成交价可能与信号价不同
   - 建议预留0.2%-0.5%滑点空间
   - 避免在涨跌停附近交易

3. **执行纪律**
   - 严格按照策略信号执行
   - 不要情绪化交易
   - 设置自动化交易系统

4. **资金管理**
   - 单笔交易不超过总资金的20%
   - 网格策略预留应急资金
   - 海龟交易严格控制单笔风险2%

### 7.3 技术风险

1. **数据质量**
   - 确保历史数据准确完整
   - 处理停牌、复权等特殊情况
   - 定期更新数据源

2. **回测偏差**
   - 避免前视偏差（look-ahead bias）
   - 考虑交易成本
   - 使用真实可交易价格

3. **系统故障**
   - 网络中断可能影响交易
   - 建议设置应急措施
   - 定期备份策略和数据

### 7.4 合规风险

- 遵守证券交易法律法规
- 不进行内幕交易
- 注意个股涨跌幅限制
- 关注监管政策变化
