## 测试步骤

### 步骤1: 启动应用
```bash
cd "D:\ai_work\stock_test"
python app_with_cache.py
```

### 步骤2: 访问页面
```
浏览器打开: http://localhost:5000
```

### 步骤3: 获取数据
1. 点击"数据管理"标签页
2. 选择板块（如"中证500"）
3. 点击"开始获取全部数据"
4. 等待数据获取完成

### 步骤4: 运行回测
1. 点击"回测"标签页
2. 选择"按板块选择"
3. 勾选一个或多个板块
4. 点击"运行回测"
5. 等待回测完成

### 步骤5: 验证展示
- 查看交易记录表格是否显示
- 确认表格有序号、代码、买入价、卖出价等字段
- 确认收益率有颜色标记

### 步骤6: 导出Excel
1. 点击"导出完整Excel"按钮
2. 等待下载提示
3. 检查浏览器下载目录
4. 打开Excel文件验证内容
