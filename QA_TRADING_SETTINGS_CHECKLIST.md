### POST /api/backtest/settings
```bash
# 测试命令
curl -X POST http://localhost:5000/api/backtest/settings \
  -H "Content-Type: application/json" \
  -d '{
    "initial_capital": 500000,
    "position_ratio": 0.333,
    "commission_rate": 0.0005,
    "slippage": 0.005
  }'

# 期望响应 (200 OK)
{
  "success": true,
  "message": "配置已更新",
  "settings": {
    "initial_capital": 500000,
    "position_ratio": 0.333,
    "commission_rate": 0.0005,
    "slippage": 0.005
  }
}
```

- [ ] 可以正确保存配置
- [ ] 参数验证正确
- [ ] 响应格式正确
