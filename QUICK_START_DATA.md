# 🚀 数据获取快速入门（30秒上手）

## ⚡ 最快开始方式

### 第1步：启动应用
```bash
python app_with_cache.py
```

### 第2步：打开浏览器
访问：**http://localhost:5000**

### 第3步：获取数据

在 **📊 数据管理** 标签页中选择：

```
板块：创业板
时间：2024-01-01 ~ 2025-02-13
点击：批量获取板块数据 ✓
```

### 第4步：等待完成
后台自动下载20只股票的历史数据到本地数据库

### 第5步：查看结果
进入 **ℹ️ 状态** 标签查看缓存数据

## 📱 三种获取方式

### 1️⃣ 单只股票（最快）
```
输入股票代码 → 点击获取数据
```
**适用**：快速测试、验证单只股票

### 2️⃣ 按板块获取（推荐）⭐
```
选择板块 → 设置时间 → 批量获取板块数据
```
**适用**：获取整个板块、策略对比

### 3️⃣ 快速预设（预设中证500）
```
点击快速获取中证500
```
**适用**：一键获取默认数据

## ❓ 遇到问题？

### 无法获取？
- ✓ 检查网络连接
- ✓ 确认Flask应用运行中
- ✓ 尝试单只股票获取测试

### 获取太慢？
- ✓ 首次获取需要下载数据（正常）
- ✓ 可减少MAX_STOCKS数量在config.py

### 想清空数据？
- ✓ 进入**ℹ️ 状态** → 点击"清空所有缓存"
