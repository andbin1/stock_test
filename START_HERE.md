## 🆘 遇到问题？

### 问题1: 显示"No module named 'akshare'"

**解决**：
```bash
pip install -r requirements.txt
```

### 问题2: 运行很慢

**原因**：首次需要下载几百只股票的历史数据

**解决**：
- 修改 `config.py` 中 `MAX_STOCKS = 30` 先测试
- 之后数据会缓存，速度会快

### 问题3: 没有交易信号

**可能原因**：
- 数据获取失败（网络问题）
- 参数条件过严格（量能倍数太高）

**解决**：
- 检查网络
- 在 `config.py` 中降低 `volume_multiplier`

查看 [USAGE.md](USAGE.md) 中的"常见错误"部分。
