## 四大功能需求

| 功能 | 参数名 | 默认值 | 范围 | 单位 | 说明 |
|------|--------|--------|------|------|------|
| 初始金额 | initial_capital | 100000 | 10k-10M | 元 | 用于计算持仓数量 |
| 交易占比 | position_ratio | 0.2 | 1%-99% | 百分比 | 每次交易占初始资金比例 |
| 手续费 | commission_rate | 0.001 | 0%-100% | 百分比 | 交易成本计算 |
| 滑点 | slippage | 0.0 | 0%-5% | 百分比 | 买入加、卖出减 |

## 关键公式

### 1. 持仓数量计算
```
position_size = floor(initial_capital * position_ratio / current_price)
```
- 按手（100股）取整
- 影响每次交易的股数

### 2. 滑点应用
```
买入价 = 原价 * (1 + slippage)
卖出价 = 原价 * (1 - slippage)
```
- 模拟市场冲击成本

### 3. 手续费计算
```
交易成本 = 交易金额 * commission_rate
```
- 应用到买卖两次
