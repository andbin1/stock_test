## 第一部分：成本模型设计

### 1.1 A股交易成本构成

```
A股实际交易成本（2024年）：

买入成本：
  手续费 = max(成交额 × 0.025%, 5元)
  滑点   = 成交额 × 0.1%

卖出成本：
  手续费 = max(成交额 × 0.025%, 5元)
  印花税 = 成交额 × 0.1%
  滑点   = 成交额 × 0.1%

总成本双边 = 0.025% + 0.1% + 0.025% + 0.1% + 0.1% = 0.35%
             (买手续费) + (买滑点) + (卖手续费) + (卖印花税) + (卖滑点)
```

## 第二部分：风险指标体系

### 2.1 核心指标速查表

| 指标 | 公式简化 | 说明 | 参考值 |
|------|--------|------|--------|
| **MaxDD** | min(回撤) | 最大风险 | -5% ～ -20% |
| **年化收益** | total_return^(252/days) | 收益效率 | 15% ～ 25% |
| **年化波动** | std(returns) × √252 | 风险大小 | 15% ～ 25% |
| **夏普比率** | (年收益-无风险)/年波动 | 风险调整收益 | 1.0 ～ 2.0 |
| **卡玛比率** | 年收益 / \|MaxDD\| | 收益/风险比 | 1.0 ～ 2.0 |
| **盈亏比** | 总盈利 / 总亏损 | 获利能力 | 1.5 ～ 3.0 |
| **胜率** | 盈利次数 / 总交易数 | 准确率 | 50% ～ 70% |
| **最大单损** | min(returns) | 最坏情况 | -5% ～ -10% |
| **索提诺** | (年收益-无风险)/下行波动 | 下行风险 | 0.7 ～ 1.5 |

### 2.2 计算示例

假设20次交易，收益序列：
```
[2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]
```

计算过程：
```
胜率 = 14/20 = 70%
平均收益 = 43%/20 = 2.15%
平均盈利 = 43%/14 = 3.07%
平均亏损 = 11%/6 = 1.83%
盈亏比 = 3.07/1.83 = 1.68

累积收益 = (1.02)(1.03)...(1.03) - 1 = 45.2%
最大回撤 = -4.14%（在第16次交易后）

如果交易期为344天（1.365年）:
  年化收益 = (1.452)^(1/1.365) - 1 = 18.06%
```

### 2.3 评分标准

```
夏普比率：
  > 2.0: A+（杰出）
  1.0-2.0: A（优秀）
  0.5-1.0: B+（良好）
  0-0.5: B（一般）
  < 0: C（需改进）

卡玛比率：
  > 2.0: 优秀
  1.0-2.0: 良好
  0.5-1.0: 一般
  < 0.5: 需改进

胜率：
  > 65%: 优秀
```

### 4.3 交易执行规则

#### 买入执行

```
信号产生: T日
执行价 = 收盘价 × (1 + 0.1%)  [买入时滑点向上]
手续费 = max(成交额×0.025%, 5元)
```

#### 卖出执行

```
信号产生: T日
执行价 = 收盘价 × (1 - 0.1%)  [卖出时滑点向下]
手续费 = max(成交额×0.025%, 5元)
印花税 = 成交额 × 0.1%
```

#### T+1规则

```
买入日T:
  - T日: 买入确认
  - T+1日: 资金冻结，无法交易
  - T+2日及以后: 可卖出

实现:
  days_elapsed = (current_date - last_trade_date).days
  can_trade = days_elapsed >= 1
```

### 4.4 边界条件

#### 涨跌停处理

```python
change = (close - prev_close) / prev_close
if abs(change) > 0.098:  # 涨跌停
```

#### 零成交量

## 常见问题解答

### Q1: 为什么要使用动态成本模型而不是固定0.1%？

**A**:
- 固定0.1%对小额交易严重低估，对大额交易高估
- 小额交易实际成本可达2%以上（触发最低费）
- 大额交易实际成本仅0.35%
- 动态模型能准确反映真实交易成本，使回测更可信

### Q2: 回测结果与实盘差异大怎么办？

**A**: 检查以下几点：
1. 成本模型是否准确（最常见问题）
2. 滑点是否合理（0.1%较保守）
3. 是否正确处理涨跌停和停牌
4. 是否有前向偏差（使用了未来数据）
5. 样本是否足够（>200笔交易）

### Q3: 夏普比率和卡玛比率如何选择？

**A**:
- **夏普比率**: 考虑所有波动，更通用
- **卡玛比率**: 关注最大回撤，风险评估更直接

建议两个都计算，综合评估策略。

### Q4: 胜率低（<50%）的策略是否可行？

**A**: 可能可行，如果：
- 盈亏比足够大（> 3:1）
- 夏普比率为正
- 最大回撤在可控范围内

例如：40%胜率 + 3倍盈亏比 = 正期望收益
