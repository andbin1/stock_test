# 参数配置K线图示功能说明

## 📈 功能概述

在参数配置页面添加了**实时K线信号预览功能**，用户调整参数时可以立即看到买卖信号如何变化。

---

## 🎯 设计目标

**解决的问题**：
- ❌ 参数调整后不知道效果如何
- ❌ 不清楚参数对信号的影响
- ❌ 需要回测才能看到结果，调参效率低

**提供的价值**：
- ✅ 实时可视化买卖信号
- ✅ 直观理解参数对策略的影响
- ✅ 快速验证参数组合是否合理
- ✅ 无需回测即可预览效果

## 📊 K线图示详解

### 显示元素

| 元素 | 颜色 | 含义 | 说明 |
|------|------|------|------|
| 🕯️ K线 | 红色/绿色 | 价格走势 | 红涨绿跌（中国习惯） |
| ━━ 粗线 | 橙色 | 主均线 | 基于 `ma_period` 参数 |
| ╌╌ 虚线 | 蓝色 | 回踩均线 | 基于 `retest_period` 参数 |
| 📊 柱状图 | 红色/绿色半透明 | 成交量 | 显示在图表底部 |
| ▲ 三角 | 红色 | 买入信号 | 满足所有买入条件 |
| ▼ 倒三角 | 绿色 | 卖出信号 | 达到持有天数 |
| ━━ 连线 | 橙色半透明 | 持有期 | 连接买卖点 |

### 信号检测逻辑

**买入条件**（模拟策略逻辑）：
1. 价格在主均线上方
2. 价格接近回踩均线（2%范围内）
3. 成交量放大（最近3天总量 > 20日均量 × 量能倍数）

**卖出信号**：
- 买入后持有指定天数自动卖出

## 🎮 使用方法

### 步骤1：打开参数配置页面

```
方式1：主页点击 "⚙️ 参数配置" 按钮
方式2：直接访问 http://localhost:5000/parameters
```

### 步骤2：观察初始图示

页面加载后，右侧会自动显示：
- 60天模拟K线数据
- 基于当前参数的均线
- 检测到的买卖信号

### 步骤3：调整参数观察变化

**实时更新**：拖动任何参数滑块，图表会在300毫秒后自动更新

**关键参数对图表的影响**：

#### 📊 均线周期 (ma_period)
```
调小 (如 10 天)：
  ✓ 均线更贴近价格
  ✓ 买入信号更频繁
  ✗ 可能误判短期波动

调大 (如 60 天)：
  ✓ 趋势更明确
  ✓ 信号更可靠
  ✗ 信号数量减少
```

#### 📈 量能倍数 (volume_multiplier)
```
调小 (如 1.5 倍)：
  ✓ 更容易触发信号
  ✓ 捕捉更多机会
  ✗ 可能出现假突破

调大 (如 3 倍)：
  ✓ 信号更强烈
  ✓ 成功率更高
  ✗ 机会减少
```

#### 🔄 回踩周期 (retest_period)
```
调小 (如 3 天)：
  ✓ 快速响应
  ✓ 适合短线
  ✗ 噪音较多

调大 (如 13 天)：
  ✓ 更稳定
  ✓ 适合中长线
  ✗ 滞后性增加
```

#### ⏱️ 持有天数 (hold_days)
```
调小 (如 2 天)：
  ✓ 快进快出
  ✓ 资金周转快
  ✗ 错过大趋势

调大 (如 10 天)：
  ✓ 充分跟随趋势
  ✓ 单次收益更高
  ✗ 资金占用时间长
```

### 步骤4：保存满意的参数

```
1. 调整到理想效果
2. 点击 "💾 保存参数"
3. 返回主页进行回测验证
```

## 💡 使用技巧

### 技巧1：观察信号密度

**信号过多**（10+ 个买入信号）：
- 说明条件太宽松
- 建议：提高量能倍数或均线周期

**信号过少**（0-2 个买入信号）：
- 说明条件太严格
- 建议：降低量能倍数或均线周期

**信号适中**（3-8 个买入信号）：
- 比较合理的参数组合
- 可以进行回测验证

### 技巧2：观察买卖点位置

**好的信号特征**：
- ✅ 买在回调低点
- ✅ 卖在上涨途中
- ✅ 持有期间价格上涨
- ✅ 避开暴跌区域

**差的信号特征**：
- ❌ 买在高位
- ❌ 卖在低位
- ❌ 持有期间价格下跌
- ❌ 在震荡区域频繁交易

### 技巧3：对比不同参数组合

```
方法：
1. 记录当前图表的信号数量和位置
2. 调整参数
3. 对比新旧信号的质量
4. 选择信号更优质的参数组合
```

### 技巧4：结合预设功能

```
流程：
1. 调整参数，观察图表
2. 发现好的组合 → 保存为预设
3. 切换其他参数继续探索
4. 对比多个预设的图表效果
5. 选择最优预设进行回测
```

## 🎨 技术实现

### 前端技术

**Canvas 2D API**：
```javascript
// 绘制K线
ctx.fillRect(x, y, width, height);

// 绘制均线
ctx.moveTo(x1, y1);
ctx.lineTo(x2, y2);
ctx.stroke();

// 绘制信号
ctx.fillText('▲', x, y);
```

**实时更新机制**：
```javascript
// 防抖更新
clearTimeout(window.chartUpdateTimer);
window.chartUpdateTimer = setTimeout(() => {
    drawKLineChart();
}, 300);  // 300ms 延迟
```

### 数据生成

**模拟K线数据**：
- 60天随机价格数据
- 符合真实股票走势规律
- 包含开高低收和成交量

**指标计算**：
- MA均线：简单移动平均
- 成交量均线：20日均量
- 买入信号：多条件检测
- 卖出信号：持有天数触发

## 📌 注意事项

### ⚠️ 重要提示

**1. 模拟数据**
```
图表使用的是模拟数据，仅用于参数调试
真实回测请使用主页的回测功能
```

**2. 信号示意**
```
图表显示的信号是基于简化的策略逻辑
实际回测可能因更多条件而有差异
```

**3. 性能考虑**
```
图表每次参数变化后延迟300ms更新
避免过于频繁的重绘影响性能
```

**4. 浏览器兼容**
```
需要支持Canvas 2D的现代浏览器
推荐：Chrome、Edge、Firefox
```
