# 回测系统核心风险指标体系

## 概述

本文档定义了量化回测系统中所有关键风险指标的计算方法、含义、依赖关系和参考值。这些指标用于评估策略的收益性、风险性和稳定性。

---

## 1. 最大回撤 (MaxDD, Maximum Drawdown)

### 1.1 基本定义

**中文名**: 最大回撤 / 最大回撤率
**英文名**: Maximum Drawdown / Max DD

**定义**: 从最高点到最低点的最大下跌幅度。是衡量风险最重要的指标。

### 1.2 数学公式

```
累积收益 = 起始资金
对于每个交易 t：
    cum_return[t] = cum_return[t-1] × (1 + return[t])

运行最高值 = max(cum_return[0...t])

回撤 = (cum_return[t] - running_max[t]) / running_max[t]

MaxDD = min(回撤) = min((cum_return[t] - running_max[t]) / running_max[t])
```

### 1.3 数值示例

```python
# 示例：20次交易的收益序列
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

# 累积收益计算
cum_return = [100]
for r in trades_return:
    cum_return.append(cum_return[-1] * (1 + r/100))

# 结果
cum_return = [100, 102, 105.06, 104, 108.16, 110.33, 107.08, 112.44, 113.56, 111.28,
              114.62, 116.91, 121.79, 120.57, 124.18, 126.66, 121.41, 128.70, 129.98, 132.58,
              136.56]

running_max = [100, 102, 105.06, 105.06, 108.16, 110.33, 110.33, 112.44, 113.56, 113.56,
               114.62, 116.91, 121.79, 121.79, 124.18, 126.66, 126.66, 128.70, 129.98, 132.58,
               136.56]

drawdown = [(cum - running_max[i]) / running_max[i]
            for i, cum in enumerate(cum_return)]

# 最大回撤在第16次交易后
# MaxDD = (121.41 - 126.66) / 126.66 = -0.0414 = -4.14%
```

### 1.4 含义解释

- **值域**: [-100%, 0)（0表示无回撤，-100%表示全部亏损）
- **数值解释**:
  - -5% ~ -10%: 很好，风控严格
  - -10% ~ -20%: 良好，风险可控
  - -20% ~ -30%: 一般，需要优化
  - > -30%: 风险较大，需要改进

### 1.5 计算依赖

- 依赖项: 所有交易收益率序列
- 时间复杂度: O(n)
- 更新频率: 每次新交易后

### 1.6 边界条件

```
1. 若没有任何亏损，MaxDD = 0
2. 若整个回测期间单调下跌，MaxDD = 最终收益率（若为负）
3. 若只有一笔交易，MaxDD = min(0, 交易收益)
4. 若初始资金为0，MaxDD = 不可定义(NaN)
```

### 1.7 参考值

| 策略类型 | 优秀(%) | 良好(%) | 一般(%) | 需改进(%) |
|---------|--------|--------|--------|----------|
| 量能突破 | > -8 | -8 ~ -12 | -12 ~ -20 | < -20 |
| 趋势跟踪 | > -10 | -10 ~ -15 | -15 ~ -25 | < -25 |
| 动量策略 | > -15 | -15 ~ -25 | -25 ~ -40 | < -40 |
| 多因子 | > -10 | -10 ~ -15 | -15 ~ -25 | < -25 |

## 2. 年化收益率 (Annual Return / Annualized Return)

### 2.1 基本定义

**中文名**: 年化收益率 / 年化回报率
**英文名**: Annual Return / Annualized Return / ARR

**定义**: 假设收益持续一整年，折合出的年度百分比收益率。

### 2.2 数学公式

```
总收益率 = (最终资金 - 初始资金) / 初始资金

交易周期(天) = 最后交易日 - 第一交易日

交易周期(年) = 交易周期(天) / 252（252为A股年均交易天数）

年化收益率 = (1 + 总收益率) ^ (1 / 交易周期(年)) - 1

或简化形式（对于短期）：
年化收益率 ≈ 总收益率 / 交易周期(年)
```

### 2.3 数值示例

```python
# 示例：从2024-01-01到2024-12-31进行回测

初始资金 = 100,000元
最终资金 = 125,000元
总收益率 = (125,000 - 100,000) / 100,000 = 0.25 = 25%

第一笔交易日期 = 2024-01-10
最后一笔交易日期 = 2024-12-20
交易周期 = 344天

交易周期(年) = 344 / 252 = 1.365年

年化收益率（复利）:
= (1 + 0.25) ^ (1 / 1.365) - 1
= 1.25 ^ 0.7326 - 1
= 1.1806 - 1
= 0.1806 = 18.06%

简化计算:
年化收益率 ≈ 25% / 1.365 = 18.32%
```

### 2.4 含义解释

- **值域**: (-100%, +∞)
- **数值解释**:
  - > 30%: 非常优秀，需谨慎验证是否过拟合
  - 20% ~ 30%: 优秀
  - 10% ~ 20%: 良好
  - 5% ~ 10%: 还可以
  - 0% ~ 5%: 一般
  - < 0%: 需要改进

### 2.5 计算依赖

- 依赖项: 初始资金、最终资金、交易时间段
- 依赖指标: 无
- 时间复杂度: O(1)

### 2.6 边界条件

```
1. 若交易周期 < 1天，年化收益率 = 0
2. 若交易周期非常长（>10年），年化收益率接近总收益率
3. 若总收益率为 -100%，年化收益率 = -100%
4. 若初始资金为0，年化收益率 = 不可定义(NaN)
```

### 2.7 参考值

| 对标指数 | 年均收益 |
|---------|--------|
| 沪深300 | 8% - 12% |
| 中证500 | 6% - 10% |
| 创业板指 | 8% - 15% |
| 无风险收益 | 3% - 4% |
| 优秀策略 | 15% - 25% |
| 杰出策略 | 25% - 50% |

## 3. 年化波动率 (Annual Volatility)

### 3.1 基本定义

**中文名**: 年化波动率 / 年化标准差
**英文名**: Annual Volatility / Annualized Volatility / Std Dev

**定义**: 交易收益率的标准差，年化表示。反映收益的波动程度。

### 3.2 数学公式

```
日均收益率 = [r1, r2, r3, ..., rn]（单笔交易收益率）

平均收益率 = mean(日均收益率) = Σri / n

日波动率 = sqrt(Σ(ri - 平均收益率)² / (n-1))

年化波动率 = 日波动率 × sqrt(252)
```

### 3.3 数值示例

```python
# 示例：20笔交易的收益序列
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

# 计算平均收益
mean_return = sum(trades_return) / len(trades_return) = 43% / 20 = 2.15%

# 计算方差
variance = Σ(ri - 2.15%)² / 19

# 详细计算每笔偏差平方
deviations = [(r - 2.15)² for r in trades_return]
# = [0.0144, 0.0072, 0.0100, 0.0342, 0.0017, 0.0258, 0.0812, 0.0013, 0.0173, 0.0072,
#    0.0017, 0.0342, 0.0100, 0.0072, 0.0017, 0.0374, 0.1489, 0.0013, 0.0017, 0.0072]

variance = sum(deviations) / 19 = 0.4581 / 19 = 0.02411

# 日波动率
daily_vol = sqrt(0.02411) = 0.1552 = 15.52%

# 年化波动率
annual_vol = 15.52% × sqrt(252) = 15.52% × 15.87 = 246.3%

# 注：这个例子中波动率很高，因为只有20笔交易
# 实际系统中应基于252个交易日数据
```

### 3.4 含义解释

- **值域**: [0, +∞)
- **数值解释**:
  - < 10%: 低波动，稳定收益
  - 10% ~ 20%: 中等波动，正常
  - 20% ~ 40%: 高波动，风险较大
  - > 40%: 超高波动，极度不稳定

### 3.5 计算依赖

- 依赖项: 所有交易收益率序列
- 依赖指标: 无
- 时间复杂度: O(n)

### 3.6 边界条件

```
1. 若所有交易收益相同，年化波动率 = 0
2. 若只有1笔交易，年化波动率 = 0（无法计算方差）
3. 若收益极端（如 +500%、-90%），波动率会很大
4. 波动率永远 >= 0
```

### 3.7 参考值

| 投资类型 | 年化波动 |
|---------|--------|
| 债券 | 2% - 5% |
| 指数基金 | 12% - 18% |
| 蓝筹股 | 15% - 25% |
| 中小盘 | 25% - 40% |
| 创业板 | 35% - 60% |
| 高频交易 | 5% - 15% |

## 4. 夏普比率 (Sharpe Ratio)

### 4.1 基本定义

**中文名**: 夏普比率 / 夏普指数
**英文名**: Sharpe Ratio / SR

**定义**: 风险调整后的收益。衡量单位风险获得的超额收益。**最重要的风险指标之一**。

### 4.2 数学公式

```
超额收益 = 年化收益率 - 无风险利率

夏普比率 = 超额收益 / 年化波动率
        = (年化收益率 - 无风险利率) / 年化波动率

其中：
  年化收益率 = (1 + 总收益率) ^ (252 / 交易天数) - 1
  无风险利率 = 3.5%（当前银行定期存款/国债利率）
  年化波动率 = 收益率标准差 × sqrt(252)
```

### 4.3 数值示例

```python
# 示例数据
年化收益率 = 20%
年化波动率 = 16%
无风险利率 = 3.5%

超额收益 = 20% - 3.5% = 16.5%

夏普比率 = 16.5% / 16% = 1.031

# 解释：每承担1%的波动，获得1.031%的超额收益
```

### 4.4 含义解释

- **值域**: (-∞, +∞)
- **数值解释**:
  - > 2.0: 杰出，风险调整收益优秀
  - 1.0 ~ 2.0: 优秀，风险调整收益很好
  - 0.5 ~ 1.0: 良好，风险调整收益可以
  - 0 ~ 0.5: 一般，风险调整收益平庸
  - < 0: 不好，低于无风险收益
  - 负值越大越差

### 4.5 计算依赖

- 依赖项: 所有交易收益率、交易时间段
- 依赖指标: 年化收益率、年化波动率、无风险利率
- 时间复杂度: O(n)

### 4.6 边界条件

```
1. 若年化波动率 = 0，夏普比率 = +∞（无法交易）
2. 若年化收益率 < 无风险利率，夏普比率 < 0
3. 若无风险利率设置不合理，会影响夏普比率
4. 波动率接近0时，夏普比率对小波动敏感
```

### 4.7 参考值

| 策略评级 | 夏普比率 |
|---------|--------|
| A+（杰出） | > 2.0 |
| A（优秀） | 1.0 - 2.0 |
| B+（良好） | 0.5 - 1.0 |
| B（一般） | 0.0 - 0.5 |
| C（需改进） | < 0 |
| 标普500 | 约 0.5 |
| 优秀对冲基金 | 1.0 - 1.5 |

## 5. 卡玛比率 (Calmar Ratio)

### 5.1 基本定义

**中文名**: 卡玛比率 / 卡玛指数
**英文名**: Calmar Ratio / Calmar Index

**定义**: 年化收益率与最大回撤的比值。衡量收益与风险的权衡。

### 5.2 数学公式

```
卡玛比率 = 年化收益率 / |最大回撤|
        = Ann_Return / |MaxDD|

其中 |MaxDD| 取绝对值（MaxDD为负数）
```

### 5.3 数值示例

```python
# 示例
年化收益率 = 18%
最大回撤 = -12%

卡玛比率 = 18% / 12% = 1.5

# 解释：每承担1%的最大风险，年化获得1.5%的收益
```

### 5.4 含义解释

- **值域**: (0, +∞)
- **数值解释**:
  - > 2.0: 优秀，收益与风险匹配完美
  - 1.0 ~ 2.0: 良好，收益与风险较均衡
  - 0.5 ~ 1.0: 一般，需要改进
  - < 0.5: 差，风险相对收益太大

### 5.5 计算依赖

- 依赖项: 年化收益率、最大回撤
- 依赖指标: Annual Return、MaxDD
- 时间复杂度: O(1)（已有上述指标）

### 5.6 边界条件

```
1. 若最大回撤 = 0（无亏损），卡玛比率 = +∞
2. 若年化收益率 < 0，卡玛比率 < 0（不合理，应淘汰）
3. 若最大回撤很小，卡玛比率对小变化敏感
```

### 5.7 参考值

| 策略等级 | 卡玛比率 |
|---------|--------|
| A（优秀） | > 2.0 |
| B（良好） | 1.0 - 2.0 |
| C（一般） | 0.5 - 1.0 |
| D（需改） | < 0.5 |

## 6. 索提诺比率 (Sortino Ratio)

### 6.1 基本定义

**中文名**: 索提诺比率 / 下行波动率风险调整收益
**英文名**: Sortino Ratio

**定义**: 与夏普比率类似，但只考虑下行波动（负收益波动），忽略上行波动。更精确的风险度量。

### 6.2 数学公式

```
下行波动率计算:
下行偏差 = [min(ri - 目标收益率, 0) for ri in returns]
下行波动率 = sqrt(mean(下行偏差²)) × sqrt(252)

索提诺比率 = (年化收益率 - 无风险利率) / 下行波动率
```

### 6.3 数值示例

```python
# 示例：20笔交易
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

# 假设目标收益率 = 0%
下行偏差 = [0, 0, -1%, 0, 0, -3%, 0, 0, -2%, 0,
           0, 0, -1%, 0, 0, -4%, 0, 0, 0, 0]

# 平方：[0, 0, 0.0001, 0, 0, 0.0009, 0, 0, 0.0004, 0,
#        0, 0, 0.0001, 0, 0, 0.0016, 0, 0, 0, 0]

下行方差 = sum(平方) / 20 = 0.0031 / 20 = 0.000155

下行波动率 = sqrt(0.000155) × sqrt(252) = 0.01245 × 15.87 = 19.75%

年化收益率 = 20%
无风险利率 = 3.5%

索提诺比率 = (20% - 3.5%) / 19.75% = 16.5% / 19.75% = 0.835
```

### 6.4 含义解释

- **值域**: (-∞, +∞)
- **数值解释**（与夏普比率类似但更严格）:
  - > 1.5: 优秀
  - 0.7 ~ 1.5: 良好
  - 0.3 ~ 0.7: 一般
  - < 0.3: 需要改进

### 6.5 计算依赖

- 依赖项: 所有交易收益率序列
- 依赖指标: 年化收益率
- 时间复杂度: O(n)

### 6.6 边界条件

```
1. 若下行波动率 = 0，索提诺比率 = +∞
2. 若所有收益都为正，下行波动率 = 0
3. 若目标收益率设置不合理，结果会偏差
```

## 7. 盈亏比 (Profit Factor)

### 7.1 基本定义

**中文名**: 盈亏比 / 利润因子
**英文名**: Profit Factor / PF

**定义**: 所有盈利交易的总收益与所有亏损交易的总亏损的比值。

### 7.2 数学公式

```
盈利交易列表 = [r for r in returns if r > 0]
亏损交易列表 = [r for r in returns if r <= 0]

总盈利 = sum(盈利交易列表)
总亏损 = abs(sum(亏损交易列表))

盈亏比 = 总盈利 / 总亏损

特殊情况：
  - 若总亏损 = 0（无亏损），盈亏比 = +∞（或定义为 999）
  - 若总盈利 = 0（无盈利），盈亏比 = 0
```

### 7.3 数值示例

```python
# 示例：20笔交易
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

# 盈利交易
profits = [2%, 3%, 4%, 2%, 5%, 1%, 3%, 2%, 4%, 3%, 6%, 1%, 2%, 3%]
总盈利 = 2+3+4+2+5+1+3+2+4+3+6+1+2+3 = 43%

# 亏损交易
losses = [-1%, -3%, -2%, -1%, -4%]
总亏损 = 1+3+2+1+4 = 11%

盈亏比 = 43% / 11% = 3.91

# 解释：每亏损1块钱，能赚3.91块钱
```

### 7.4 含义解释

- **值域**: [0, +∞)
- **数值解释**:
  - > 3.0: 优秀，亏损控制很好
  - 2.0 ~ 3.0: 很好，盈利能力强
  - 1.0 ~ 2.0: 良好，盈利能力可以
  - 0.5 ~ 1.0: 一般，需要改进
  - < 0.5: 差，亏损大于盈利

### 7.5 计算依赖

- 依赖项: 所有交易收益率序列
- 依赖指标: 无
- 时间复杂度: O(n)

### 7.6 边界条件

```
1. 若所有交易都盈利，盈亏比 = +∞
2. 若所有交易都亏损，盈亏比 = 0
3. 若没有任何交易，盈亏比 = 未定义
```

### 7.7 参考值

| 等级 | 盈亏比 |
|------|--------|
| A | > 2.5 |
| B | 1.5 - 2.5 |
| C | 1.0 - 1.5 |
| D | 0.5 - 1.0 |
| F | < 0.5 |

## 8. 胜率 (Win Rate)

### 8.1 基本定义

**中文名**: 胜率 / 成功率
**英文名**: Win Rate / Success Rate / WR

**定义**: 盈利交易数量占总交易数量的百分比。衡量策略的准确性。

### 8.2 数学公式

```
胜利交易数 = count([r for r in returns if r > 0])
总交易数 = len(returns)

胜率 = 胜利交易数 / 总交易数 × 100%

败率 = (总交易数 - 胜利交易数) / 总交易数 × 100%
      = 1 - 胜率
```

### 8.3 数值示例

```python
# 示例
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

胜利交易数 = 14（正收益的交易）
总交易数 = 20

胜率 = 14 / 20 × 100% = 70%
败率 = 6 / 20 × 100% = 30%
```

### 8.4 含义解释

- **值域**: [0%, 100%]
- **数值解释**:
  - > 65%: 优秀，策略准确性好
  - 55% ~ 65%: 良好，策略相对可靠
  - 45% ~ 55%: 一般，需要观察
  - < 45%: 差，策略准确性不够

### 8.5 计算依赖

- 依赖项: 所有交易收益率序列
- 依赖指标: 无
- 时间复杂度: O(n)

### 8.6 边界条件

```
1. 若没有交易，胜率 = 未定义 / 0%
2. 若所有交易盈利，胜率 = 100%
3. 若所有交易亏损，胜率 = 0%
```

### 8.7 参考值

| 策略类型 | 健康胜率 |
|---------|--------|
| 高频交易 | 55% - 70% |
| 中频交易 | 45% - 65% |
| 低频交易 | 40% - 60% |
| 趋势跟踪 | 40% - 55% |
| 均值回复 | 60% - 75% |
| 对冲基金 | 55% - 70% |

**注意**: 高胜率不一定意味着高收益，需要结合盈亏比考虑。

## 9. 最大单笔亏损 (Max Single Loss)

### 9.1 基本定义

**中文名**: 最大单笔亏损 / 单笔最大亏损
**英文名**: Max Single Loss / Worst Trade / Max Loss

**定义**: 所有交易中收益率最小（最亏）的那一笔。衡量最坏情况下的风险。

### 9.2 数学公式

```
最大单笔亏损 = min(returns) = min([r1, r2, ..., rn])

若最大单笔亏损 > 0（所有交易盈利），则最大单笔亏损 = 0
```

### 9.3 数值示例

```python
# 示例
trades_return = [2%, 3%, -1%, 4%, 2%, -3%, 5%, 1%, -2%, 3%,
                 2%, 4%, -1%, 3%, 2%, -4%, 6%, 1%, 2%, 3%]

最大单笔亏损 = min(trades_return) = -4%

# 解释：最坏的一笔交易亏损了4%
```

### 9.4 含义解释

- **值域**: [-100%, 0]
- **数值解释**:
  - > -5%: 很好，最坏情况控制严格
  - -5% ~ -10%: 良好，风险控制得当
  - -10% ~ -20%: 一般，存在较大亏损
  - < -20%: 差，单笔亏损过大

### 9.5 计算依赖

- 依赖项: 所有交易收益率序列
- 依赖指标: 无
- 时间复杂度: O(n)

### 9.6 边界条件

```
1. 若所有交易都盈利，最大单笔亏损 = 0
2. 若只有一笔交易，最大单笔亏损 = 该交易收益率
3. 最大单笔亏损永远 <= 0
```

### 9.7 参考值

| 风险等级 | 最大单笔亏损 |
|---------|------------|
| 低风险 | > -3% |
| 中风险 | -3% ~ -8% |
| 中高风险 | -8% ~ -15% |
| 高风险 | -15% ~ -25% |
| 极高风险 | < -25% |

## 10. 附加指标

### 10.1 平均盈利交易 (Average Profit)

```
平均盈利 = mean([r for r in returns if r > 0])
```

### 10.2 平均亏损交易 (Average Loss)

```
平均亏损 = abs(mean([r for r in returns if r <= 0]))
```

### 10.3 单笔最大盈利 (Max Single Profit)

```
单笔最大盈利 = max(returns)
```

### 10.4 交易数量 (Number of Trades)

```
交易数量 = len(returns)
```

### 10.5 平均持仓天数 (Average Holding Days)

```
平均持仓 = mean([卖出日期 - 买入日期 for 所有交易])
```

## 11. 综合评分系统

建议使用加权综合评分来整体评估策略：

```python
综合评分 = (
    0.25 × 年化收益率得分 +
    0.20 × 夏普比率得分 +
    0.20 × 卡玛比率得分 +
    0.15 × 盈亏比得分 +
    0.10 × 胜率得分 +
    0.10 × 最大回撤得分
)
```

其中，各指标得分 = (指标值 - 基准值) / (上限值 - 基准值) × 100，范围 [0, 100]。

## 12. 实现参考

```python
class MetricsCalculator:
    """计算所有关键指标"""

    def __init__(self, trades, initial_capital=100000, risk_free_rate=0.035):
        self.trades = trades
        self.initial_capital = initial_capital
        self.risk_free_rate = risk_free_rate
        self.returns = [t['收益率%'] / 100 for t in trades]

    def max_drawdown(self):
        """最大回撤"""
        cum_return = self._cumulative_return()
        running_max = np.maximum.accumulate(cum_return)
        drawdown = (cum_return - running_max) / running_max
        return drawdown.min()

    def annual_return(self):
        """年化收益"""
        total_return = np.prod([1 + r for r in self.returns]) - 1
        days = (self.trades[-1]['卖出日期'] - self.trades[0]['买入日期']).days
        years = days / 252
        return (1 + total_return) ** (1 / years) - 1

    def annual_volatility(self):
        """年化波动率"""
        daily_vol = np.std(self.returns)
        return daily_vol * np.sqrt(252)

    def sharpe_ratio(self):
        """夏普比率"""
        excess_return = self.annual_return() - self.risk_free_rate
        return excess_return / self.annual_volatility()

    def calmar_ratio(self):
        """卡玛比率"""
        return self.annual_return() / abs(self.max_drawdown())

    def profit_factor(self):
        """盈亏比"""
        total_profit = sum([r for r in self.returns if r > 0])
        total_loss = abs(sum([r for r in self.returns if r <= 0]))
        return total_profit / total_loss if total_loss > 0 else float('inf')

    def win_rate(self):
        """胜率"""
        wins = len([r for r in self.returns if r > 0])
        return wins / len(self.returns) * 100 if self.returns else 0

    def max_loss(self):
        """最大单笔亏损"""
        return min(self.returns) * 100 if self.returns else 0
```
